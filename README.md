# Proyectos-Tripleten-
En este repositorio encontraran evidencia de mis proyectos realizados en la plataforma tripleten (El proyecto 7 se encuentra en otro repositorio dentro de mi perfil de github con el nombre Proyecto 7) espero que puedan recisarlos y darme su opinion gracias.

En el apartado del proyecto 1 se encuentran datos sucios sobre una tienda que cuenta con diversas áreas de interés el objetivo de este proyecto es realizar una limpieza corrección de los datos. 

En el apartado del proyecto 2 se encuentran datos parecidos a los del proyecto uno, el objetivo de este proyecto es realizar limpieza de los datos y filtrar los datos para verificar áreas específicas.

En el proyecto 3 realizo un análisis de datos sobre una plataforma de música en streming en dos ciudades diferentes en el cual filtro y verifico datos para realizar análisis y poder dar sugerencias al respecto.

En el apartado del proyecto 4 realizo un análisis sobre una tienda de tipo delivery la cual cuenta con datos como ordenes, productos y tiempos de orden en el cual analizo estos datos mediante funciones y filtros para poder realizar recomendaciones adecuadas. 

En el proyecto 5 realizó un análisis sobre ventas en una tienda de planes telefónicos en el cual se realizan filtros y funciones que funcionan para verificar datos específicos y poder dar un análisis de datos y recomendaciones al respecto.

En el proyecto 6 realizo un análisis sobre una tienda de videojuegos la cual vende diferentes juegos para diferentes plataformas en la cual realizo una limpieza de datos, análisis y visualización(graficas) además de una formulación de hipótesis la cual verifico.

En el proyecto 8 realizo un análisis simple sobre un conjunto de datos de viajes de taxis, en este se encuentra un filtrado de datos, visualización y prueba de hipótesis.
